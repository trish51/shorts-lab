One-time setup — click for instructions (~10 min)
1. Go to console.cloud.google.com and create a free project.
2. Go to APIs & Services → Library, enable YouTube Data API v3 and YouTube Analytics API.
3. Go to APIs & Services → OAuth consent screen. Choose External, fill in the basics, add your own email as a test user.
4. Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID. Choose Web application.
5. Under Authorized JavaScript origins, add the origin shown in your browser's address bar right now for this page.
6. Copy the Client ID (ends in .apps.googleusercontent.com) and paste it below. It's not a secret — safe to store.
